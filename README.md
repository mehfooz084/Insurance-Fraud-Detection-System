# 🛡️ Insurance Fraud Detection System

A Machine Learning-based web application that predicts whether an insurance claim is **Fraudulent** or **Non-Fraudulent** using a trained Decision Tree Classifier. The project demonstrates an end-to-end ML workflow, including data preprocessing, feature engineering, model training, evaluation, and deployment with Streamlit.

---

## 🚀 Live Demo

> **https://insurance-fraud-detection-system-xycevsajzqrq6kqqpe8ude.streamlit.app/**

---

## 📌 Features

- Predicts fraudulent insurance claims using Machine Learning.
- Interactive Streamlit web application.
- Random sample claim loader for quick testing.
- Editable claim details before prediction.
- Clean and user-friendly interface.
- Model comparison dashboard.
- Prediction summary for each claim.
